# My Portfolio

A personal portfolio website to showcase my background, skills, and projects.

## Live Demo

View the portfolio here: https://ndavid-arch.github.io/My-Portfolio/

## Overview

This project is a static portfolio built with HTML, CSS, and JavaScript. It includes:

- Desktop and mobile navigation
- Profile and About sections
- Experience section with skill cards
- Projects section with GitHub and live demo links
- Contact section and footer navigation

## Tech Stack

- HTML5
- CSS3
- JavaScript (vanilla)

## Project Structure

```text
My-Portfolio/
|- index.html
|- style.css
|- mediaqueries.css
|- script.js
|- assets/
```

## Getting Started

1. Clone this repository.
2. Open the project folder.
3. Open `index.html` in your browser.

## Development Notes

- Main styles are in `style.css`.
- Responsive behavior is in `mediaqueries.css`.
- Hamburger menu logic is in `script.js`.

## Customization

- Update personal information in `index.html`.
- Replace images and icons in the `assets/` folder.
- Update project links in the Projects section.
- Adjust breakpoints and layout behavior in `mediaqueries.css`.
